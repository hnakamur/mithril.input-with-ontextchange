mithril.text-with-ontextchange two way binding example

Open the JavaScript console and input some texts in the input text and the text area below.

Logs are printed when texts change, but not between compositionstart and compositionend events.

Input app.vm.title() or app.vm.description() to see current values.

Input app.vm.title('some title') or app.vm.description('some description'), followed by m.redraw() to update values.
